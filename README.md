# Imran Ali - Professional Portfolio

This is the source code for my professional portfolio. 

## About Me
Customer Support Associate with 3+ years of experience in team leadership and quality auditing. Proficient in Salesforce, Zendesk, Jira, and remote support tools.

## Tech Stack
- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel / Netlify

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

## Configuration
To update personal information and links, modify `src/user.config.ts`.
To update resume sections, modify `src/components/utils/ResumeData.ts`.
